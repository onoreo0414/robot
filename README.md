# robot
自作ロボットの制御に関する制作物および関連資料をまとめたリポジトリである。

## 概要
本リポジトリには、自作ロボットの設計および制御に関する資料、ならびに大会出場時に使用した実装コードを掲載している。

## 資料について
自作ロボットの構成や設計方針、制御方法については、「docs/自作ロボット詳細.pdf」にまとめている。  
本制作物の全体像を把握するため、まずはこちらの資料を確認していただきたい。

また、2014年当時の大会概要および大会結果を示したPDF資料、ならびに実際に全国大会で使用した制御コードを併せて掲載している。

## 動画資料
YouTube上に、当時出場した京滋奈ブロック大会決勝戦の様子を公開している。  
実機による動作の様子や競技中の挙動を確認できるため、併せて参照していただければ幸いである。
